3-rail shell

This page is the canonical demo for the .content > nav.leftnav rail next to .content > aside. Three sticky rails on desktop; on viewports below 720px both rails stack above <main> — left nav first, aside second.

Markup

<div class="content">
  <nav class="leftnav">…</nav>
  <main>…</main>
  <aside>…</aside>
</div>

Notes

  • Use .rail-section for grouped lists in either rail.
  • Section heading: <h3 class="eyebrow">.
  • For collapsible sections, wrap in <details> + <summary class="rail-section-summary"> and load rail.js to persist state.
